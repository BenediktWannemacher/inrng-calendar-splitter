# Cycling Race Calendar Splitter

This project generates filtered iCal calendar feeds based on the INRNG Pro Cycling Calendar.

The original calendar contains many professional road cycling races in one feed. This project splits that source into smaller calendars, for example men’s WorldTour, women’s WorldTour, ProSeries, class 1 races, one-day races, and stage races.

The generated calendars are published as static `.ics` files through GitHub Pages and updated automatically with GitHub Actions.

Source calendar: https://inrng.com/calendar/

Used with permission from The Inner Ring. Please refer to the original source for updates and corrections.

This project is an independent filtered version and is not affiliated with INRNG.
